# One Page de Escritório de Advocacia

## Descrição

Este projeto é uma **página única (one page)** desenvolvida para um escritório de advocacia, utilizando **HTML**, **CSS** e **Bootstrap**. A página é projetada para ser clara, profissional e responsiva, refletindo a imagem de um escritório jurídico de alta qualidade.

## Recursos

- **HTML5**: Estrutura básica e semântica da página.
- **CSS3**: Estilos personalizados para garantir um design atraente e funcional.
- **Bootstrap**: Framework utilizado para garantir a responsividade e a consistência visual em diferentes dispositivos e tamanhos de tela.
